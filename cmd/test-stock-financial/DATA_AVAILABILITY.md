# Tushare Stock Financial API 数据可用性说明

## 📊 数据可用性概览

### ✅ 通常有丰富数据的 API (高可用性)

这些 API 历史数据完整，几乎所有上市公司都有数据：

1. **Balancesheet (资产负债表)** ⭐⭐⭐⭐⭐
   - 数据完整性：非常高
   - 覆盖范���：所有 A 股上市公司
   - 时间范围：通常从上市开始到现在
   - 更新频率：每季度更新
   - 注意事项：需要提供 ts_code 参数

2. **Income (利润表)** ⭐⭐⭐⭐⭐
   - 数据完整性：非常高
   - 覆盖范围：所有 A 股上市公司
   - 时间范围：通常从上市开始到现在
   - 更新频率：每季度更新
   - ��意事项：需要提供 ts_code 参数

3. **Cashflow (现金流量表)** ⭐⭐⭐⭐⭐
   - 数据完整性：非常高
   - 覆盖范围：所有 A 股上市公司
   - 时间范围：通常从上市开始到现在
   - 更新频率：每季度更新
   - 注意事项：需要提供 ts_code 参数

### ⚠️ 数据取决于公司行为的 API (中等可用性)

这些 API 的数据取决于公司是否有相关行为：

4. **Dividend (分红送股)** ⭐⭐⭐⭐
   - 数据完整性：取决于公司分红政策
   - 覆盖范围：有分红历史的公司
   - 注意事项：
     - 不是所有公司都分红
     - 银行、成熟企业分红较多
     - 成长型公司可能不分红

5. **FinaIndicator (财务指标)** ⭐⭐⭐⭐
   - 数据完整性：高
   - 覆盖范围：所有上市公司
   - 包含：ROE、ROA、负债率等关键指标
   - 注意事项：部分指标对于新上市公司可能缺失

6. **FinaAudit (财务审计意见)** ⭐⭐⭐⭐
   - 数据完整性：高
   - 覆盖范围：所有上市公司
   - 数据类型：审计意见（标准无保留、带强调事项等）
   - 注意事项：每年更新一次

### 📅 季节性/时效性数据的 API (波动性)

这些 API 的数据有很强的时效性：

7. **DisclosureDate (财报披露日期)** ⭐⭐⭐
   - 数据完整性：取决于财报季节
   - 覆盖范围：所有上市公司
   - 最佳查询时间：
     - 1-4月：年报披露期
     - 4-8月：一季报和半年报
     - 7-10月：三季报
   - 注意事项：非财报季节数据较少

8. **Express (业绩快报)** ⭐⭐
   - 数据完整性：较低
   - 覆盖范围：部分公司
   - 发布时间：财报正式披露前
   - 注意事项：
     - 不是所有公司都发布业绩快报
     - 通常在年报和半年报期间有数据

9. **Forecast (业绩预告)** ⭐⭐⭐
   - 数据完整性：中等
   - 覆盖范围：部分公司
   - 发布时间：财报正式披露前
   - 注意事项：
     - 主要在业绩变动较大时发布
     - 类型包括：预增、预减、扭亏、首亏等

### 🏢 特定条件的 API (条件性)

10. **FinaMainbz (主营业务构成)** ⭐⭐⭐
    - 数据完整性：中等
    - 覆盖范围：多元化公司
    - 数据类型：
      - P：按产品分类
      - D：按地区分类
    - 注意事项：
      - 单一业务公司数据较少
      - 大型多元化公司数据丰富

## ⚡ API 限制说明

### 积分要求
- 所有财务 API 都需要一定的积分权限
- 具体要求请参考：https://tushare.pro/document/2?doc_id=16

### 调用频率限制
- 单次调用通常返回多条记录
- 建议按时间范围分段查询大量数据
- 避免频繁调用相同参数

### 数据更新频率
- 季度财报：每季度更新
- 年报：每年更新
- 业绩预告/快报：不定期

## 🔧 调试建议

如果某个 API 返回空数据：

1. **检查参数是否正确**
   - ts_code 格式：000001.SZ 或 600000.SH
   - 日期格式：20240101
   - period 格式：20241231

2. **尝试不同的时间范围**
   - 查询最近季度：Period = "20240930"
   - 查询去年：StartDate = "20230101"

3. **尝试不同的股票**
   - 使用推荐的测试股票
   - 避免使用 ST 股或退市股

4. **检查权限**
   - 确认 Token 积分足够
   - 访问 Tushare 官网查看积分等级

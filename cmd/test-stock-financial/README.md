# Tushare Stock Financial API 测试

这个测试程序用于测试 tushare-go 项目中的所有财务相关 API (stock_financial)，并提供**可视化图表展示**功能。

## ✨ 特性

- 📊 **图表展示** - 可视化展示各 API 的返回数据情况
- 📋 **详细报告** - 生成包含统计数据的详细报告
- 🌐 **HTML报告** - 自动生成美观的 HTML 可视化报告
- 🎯 **多模式** - 支持简单文本模式和图表展示模式

## 测试的 API 列表

本测试程序测试以下 10 个财务 API：

1. **Balancesheet** - 获取上市公司资产负债表
2. **Cashflow** - 获取上市公司现金流量表
3. **DisclosureDate** - 获取财报披露计划日期
4. **Dividend** - 分红送股数据
5. **Express** - 获取上市公司业绩快报
6. **FinaAudit** - 获取上市公司定期财务审计意见数据
7. **FinaIndicator** - 获取上市公司财务指标数据
8. **FinaMainbz** - 获得上市公司主营业务构成
9. **Forecast** - 获取业绩预告数据
10. **Income** - 获取上市公司财务利润表数据

## 🚀 快速开始

### 1. 设置环境变量

```bash
export TUSHARE_TOKEN="your_token_here"
```

### 2. 运行测试

#### 方式1: 简单文本模式 (推荐新手)

```bash
TUSHARE_TOKEN="your_token" go run main.go
```

#### 方式2: 图表展示模式 ⭐ 推荐

```bash
TUSHARE_TOKEN="your_token" go run main_chart.go
```

#### 方式3: 使用脚本

```bash
# 简单模式
TUSHARE_TOKEN="your_token" ./run-test.sh

# 图表模式
TUSHARE_TOKEN="your_token" ./run-test.sh chart

# 同时运行两种模式
TUSHARE_TOKEN="your_token" ./run-test.sh both
```

## 📝 数据可用性说明

根据我们的测试，各 API 的数据可用性如下：

### ⭐⭐⭐⭐⭐ 数据丰富 (几乎总是有数据)
- Balancesheet (资产负债表)
- Income (利润表)
- Cashflow (现金流量表)

### ⭐⭐⭐⭐ 取决于公司行为
- Dividend (分红) - 分红公司才有
- FinaIndicator (财务指标) - 大部分公司有
- FinaAudit (审计意见) - 每年更新

### ⭐⭐⭐ 季节性或特定条件
- DisclosureDate (披露日期) - 财报季节数据多
- FinaMainbz (主营业务构成) - 多元化公司数据多
- Forecast (业绩预告) - 业绩波动时发布

### ⭐⭐ 较少发布
- Express (业绩快报) - 非所有公司发布

详细说明请参考 [DATA_AVAILABILITY.md](./DATA_AVAILABILITY.md)

## 🛠️ 故障排除

### 问题1: Token 权限不足

```
错误: API调用失败: 权限不足
```

**解决方案**：
1. 访问 https://tushare.pro 查看你的积分等级
2. 某些财务 API 需要特定积分权限
3. 考虑升级账户或使用免费 API

### 问题2: 返回空数据

```
⚠️  成功但无数据
```

**这是正常现象**，原因：
- 该时间段公司没有相关业务行为
- Express/Forecast 不是所有公司都发布
- 尝试其他股票或时间范围

### 问题3: 网络错误

```
错误: 请求超时
```

**解决方案**：
1. 检查网络连接
2. 尝试使用 VPN
3. 稍后重试（可能是服务器临时问题）

## 使用建议

### 定期监控
```bash
# 每周运行一次，保存 HTML 报告
TUSHARE_TOKEN="xxx" ./run.sh
mv test_report.html "report_$(date +%Y%m%d).html"
```

### 对比不同股票
```bash
# 测试多只股票
for stock in "000001.SZ" "600519.SH" "600000.SH"; do
    # 修改代码中的 ts_code 为 $stock
    TUSHARE_TOKEN="xxx" ./run.sh
    mv test_report.html "report_${stock}.html"
done
```

## 故障排除

### 问题1: 终端图表显示异常

**症状**: 柱状图不对齐或显示乱码

**解决方案**:
1. 确保终端宽度足够（建议 80 字符以上）
2. 使用支持 UTF-8 的终端
3. 或直接查看 HTML 报告

### 问题2: HTML 报告无法打开

**症状**: 浏览器无法打开 test_report.html

**解决方案**:
1. 检查文件是否生成: `ls -la test_report.html`
2. 使用绝对路径打开
3. 尝试其他浏览器

### 问题3: 数据为 0

**症状**: 某些 API 返回 0 条记录

**这是正常现象**:
- Express: 不是所有公司都发布业绩快报
- Forecast: 业绩波动时才发布预告
- DisclosureDate: 取决于查询时间

## 技术细节

### 终端图表实现
- 使用 `strings.Repeat("█", length)` 生成柱状图
- 根据最大值计算相对长度
- 适配终端宽度（最大 50 个字符）

### HTML 报告实现
- 使用纯 HTML + CSS
- 渐变色彩设计
- 响应式布局
- 无需外部依赖

### 统计计算
- 成功率 = 成功数 / 总数 * 100%
- 空数据率 = 空数据数 / 总数 * 100%
- 总记录数 = 所有 API 返回记录数之和
